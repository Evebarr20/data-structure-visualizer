Stack Details

Time Complexity

- Push Operation: Adding an element to the top of the stack. This operation typically has a time complexity of O(1) (constant time). It doesn't depend on the number of elements already in the stack.

- Pop Operation: Removing an element from the top of the stack. Similar to the push operation, popping an element from the stack also has a time complexity of O(1). It takes constant time, regardless of the size of the stack.

- Peek Operation: Viewing the top element of the stack without removing it. This operation also has a time complexity of O(1). It involves directly accessing the top element of the stack, which takes constant time.

- Clear Operation: Clearing or emptying the stack. Typically, the clear operation is also O(1). It involves resetting the stack to its initial empty state, which doesn't depend on the number of elements in the stack.

Common implementations

- Browser History: Stacks are employed in web browsers to manage the back and forward functionality. Each visited webpage is pushed onto the stack, enabling users to navigate backward and forward through their browsing history.

- Undo Functionality: Stacks are used to implement undo functionality in applications like text editors and graphic design software. Each action that modifies the state (e.g., typing, deleting) is pushed onto the stack, allowing users to undo those actions sequentially.

More about Stacks

The concept of a stack originated from the notion of a "pushdown list" introduced by Alan Turing in 1946. Turing used this concept in his theoretical model of computation known as Turing machines. However, the stack as a data structure wasn't formally defined until around 1957

// Define the maximum size of the stack
#define MAX_SIZE 100

// Define the structure for the stack
struct Stack {
    int arr[MAX_SIZE];  // Array to store stack elements
    int top;            // Index of the top element in the stack
};

// Method to initialize the stack
void initialize(struct Stack *stack) {
    stack->top = -1;  // Initialize the top index to -1 (indicating an empty stack)
}

// Method to check if the stack is empty
int isEmpty(struct Stack *stack) {
    return stack->top == -1;  // Returns 1 if the stack is empty, 0 otherwise
}

// Method to push an element onto the stack
void Push(struct Stack *stack, int element) {
    if (stack->top == MAX_SIZE - 1) {  // Check if the stack is full
        printf("Stack overflow\n");    // Print an error message if the stack is full
        return;
    }
    stack->arr[++stack->top] = element;  // Increment top and add the element to the stack
}

// Method to remove the top element (peek) of the stack
int Pop(struct Stack *stack) {
    if (isEmpty(stack)) {                // Check if the stack is empty
        printf("Stack underflow\n");     // Print an error message if the stack is empty
        exit(1);                          
    }
    return stack->arr[stack->top--];      // Decrement top and return the popped element
}

// Method to get the top element of the stack without removing it
int Peek(struct Stack *stack) {
    if (isEmpty(stack)) {                // Check if the stack is empty
        printf("Stack is empty\n");      // Print an error message if the stack is empty
        exit(1);                          
    }
    return stack->arr[stack->top];       // Return the top element of the stack
}

// Method to get the size of the stack
int getSize(struct Stack *stack) {
    return stack->top + 1;  // Return the number of elements in the stack
}

// Method to clear the stack (top to bottom)
void clear(struct Stack *stack) {
    stack->top = -1;  // Reset the top index to -1 (indicating an empty stack)
}

int main() {
    struct Stack stack;  // Declare a stack variable
    initialize(&stack);  // Initialize the stack

    Push(&stack, 1);  // Push elements onto the stack
    Push(&stack, 2);
    Push(&stack, 3);

    printf("Peek: %d\n", Peek(&stack));        // Print the top element of the stack
    printf("Size: %d\n", getSize(&stack));     // Print the size of the stack

    Pop(&stack);  // Pop an element from the stack
    printf("Size after Pop: %d\n", getSize(&stack));  // Print the size of the stack after popping

    clear(&stack);  // Clear the stack
    printf("Size after clear: %d\n", getSize(&stack));  // Print the size of the stack after clearing

    return 0;
}

# Stack class
class Stack:
    def __init__(self):
        # Initialize the stack with an empty list
        self.arr = []
        
    # Method to check if the stack is empty
    def is_empty(self):
        # Check if the stack is empty
        return len(self.arr) == 0
    
    # Method to push a value onto the stack
    def push(self, element):
        # Add an element to the top of the stack
        self.arr.append(element)
    
    # Method to pop the top element of the stack
    def pop(self):
        # Remove and return the top element from the stack
        if self.is_empty():
            # Check if the stack is empty
            print("Stack underflow")  # Print error message
            exit(1)  
        return self.arr.pop()  # Pop and return the top element
    
    # Method to check the top (peek) of the stack
    def peek(self):
        # Return the top element of the stack without removing it
        if self.is_empty():
            # Check if the stack is empty
            print("Stack is empty")  # Print error message
            exit(1) 
        return self.arr[-1]  # Return the top element
    
    # Method to check the size of the stack
    def get_size(self):
        # Return the number of elements in the stack
        return len(self.arr)
    
    # Method to clear the stack of all elements (top to bottom)
    def clear(self):
        # Clear all elements from the stack
        self.arr = []

if __name__ == "__main__":
    # Create an instance of the Stack class
    stack = Stack()

    stack.push(1)

    stack.push(3)
    
    # Peek at the top element of the stack and print it

    # Get the size of the stack and print it

    # Pop an element from the stack
    stack.pop()
    # Get the size of the stack after popping and print it
